Semantic HTML

Name Description
<header>Defines a page or section header
<nav>Defines navigation links
<main>Represents main content of a document
<section>Defines a section in a document
<article>Defines independent, self-contained content
<aside>Defines content aside from main content
<footer>Defines a footer for a document or section
<figure>Wraps content like images with a caption
<figcaption>Caption for a <figure> element
<mark>Highlights text
<time>Represents a specific period or time
